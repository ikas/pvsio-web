Module: GaugeSport

This module helps you building gauge widgets using SVG files. Uses the Pointer module to draw pointers for the gauges.

Parameters:
Name Type Description
id String The id of the widget instance.
coords Object The four coordinates (top, left, width, height) of the display, specifying the left, top corner, and the width and height of the (rectangular) display. Default is { top: 0, left: 0, width: 250, height: 250 }.
opt Object Options:
  • parent (String): the HTML element where the display will be appended (default is "body").
  • position (String): value for the CSS property position (default is "absolute").
  • style (String): a valid style identifier (default is "tachometer").
  • z-index (String): value for the CSS property z-index (if not provided, no z-index is applied).

    getStyleConfigs(style_id) → {Object}

    Returns the style configurations for the provided style identifier. The possible styles for the GaugeSport widget are:

  • speedomter, speedomter2, speedomter3, speedomter4, speedomter5, speedomter6, speedomter7, speedomter8
